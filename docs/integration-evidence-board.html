Definicja

Co to jest regresja?

Zmiana, która usuwa działającą, widoczną dla jury funkcję albo przywraca naprawiony błąd — nawet jeśli commit jest świeższy i wygląda jak „poprawka UI”.

Świeży commit może być clown-grade. Gałąź feat/ai-deep-dive-risk rozeszła się ze starego dashboardu — więc jej „poprawki UI” nie wiedzą nic o Data Vault, eksportach, redakcji aliasu ani o naprawie bramki AI. Wmerge’owanie ich na ślepo = ciche skasowanie tej pracy. Dlatego: triage per-plik, nie merge gałęzi.
